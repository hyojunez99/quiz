# JSON 기반 퀴즈 웹 애플리케이션

## 프로젝트 한 줄 요약
JSON 데이터 기반으로 퀴즈 문제를 관리하고,  
카테고리 선택부터 결과 화면까지 **페이지 흐름을 구성한 퀴즈 웹 애플리케이션**입니다.

---

## 진행 기간
- 2025.11.24 ~ 2025.11.27 (3일)

---

## 주요 기능 & 구현 포인트
- JSON 데이터 기반 퀴즈 문제 구성  
- 카테고리 선택 기능  
- 퀴즈 진행 로직 및 점수 계산  
- 결과 화면 출력 및 재시작 기능  
- 카테고리별 UI 컬러 적용  

---

## 기술 스택
- React  
- JSON  
- CSS3  

---

## Trouble Shooting

### 문제 1
카테고리 선택과 상관없이  
전체 퀴즈 데이터가 그대로 렌더링되는 문제가 발생했습니다.

### 해결
JSON 데이터에서 **선택한 카테고리와 일치하는 문제만 `filter`로 추출**하여  
별도의 상태로 관리하고,  
해당 데이터만 화면에 전달하도록 데이터 흐름을 수정했습니다.

---

### 문제 2
퀴즈 진행 단계별 화면 전환 기준이 명확하지 않아  
카테고리 선택 화면과 결과 화면이 동시에 노출되는  
UI 흐름 문제가 발생했습니다.

### 해결
- 카테고리 선택 여부  
- 퀴즈 완료 여부  

를 기준으로 화면 조건을 다시 정리하여,  
각 단계에서 보여줄 화면이 **명확하게 분리**되도록 구조를 수정했습니다.

---

## 개발 후 느낀 점
JSON 데이터를 그대로 사용하는 것보다  
**화면 흐름에 맞게 데이터를 가공하고 상태로 관리하는 것의 중요성**을 느꼈습니다.  

또한 기능이 늘어날수록  
화면 전환 기준을 명확히 정리하지 않으면  
UI 흐름이 쉽게 복잡해질 수 있다는 점을 배우게 되었습니다.
